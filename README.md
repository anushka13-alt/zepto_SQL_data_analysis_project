# zepto_SQL_data_analysis_project

##  Overview

This project focuses on analyzing Zepto’s product and inventory dataset using **SQL** to extract meaningful business insights. The goal is to simulate real-world data analysis tasks such as pricing analysis, inventory tracking, and category-level performance evaluation.


##  Dataset

* The dataset contains information about:

  * Product names
  * Categories
  * Prices (MRP & discounted price)
  * Discount percentages
  * Availability / stock status

* It represents a typical quick-commerce (e.g., Zepto) inventory dataset.

##  Tools & Technologies

* MySQL – Data querying and analysis
* SQL – Joins, aggregations, filtering
* Excel / CSV – Dataset handling

##  Project Workflow

 1. Data Loading

* Imported dataset into MySQL database
* Created structured tables for analysis

 2. Data Cleaning

* Checked for null values
* Standardized column formats
* Ensured data consistency

 3. Exploratory Data Analysis (EDA)

* Explored product distribution across categories
* Analyzed pricing and discount trends
* Identified stock availability patterns

 4. SQL Analysis

Performed multiple business-driven queries such as:

* Top discounted products
* Category-wise product count
* Price comparison (MRP vs selling price)
* Out-of-stock product analysis
* High-value inventory insights



##  Key Insights

* Certain categories offer higher discounts, indicating competitive pricing strategies
* A significant number of products are frequently out of stock, impacting sales potential
* Price variation across categories highlights target customer segmentation
* High-MRP products contribute significantly to inventory value
